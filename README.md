# LittleCbt

[讯飞开放平台][讯飞开放平台]

## 快速开始

### 开发

```bash
$ npm i
$ npm run dev
```

### 部署

```bash
$ npm start
$ npm stop
```

### 日志

- `spark.cbtlog` 星火大模型运行日志
- `sys.cbtlog` 系统运行日志
